Thrity Network

A free, lightweight browser and network.

no ads no tracking no accounts open source
Download thrity-network0.03.tar.gz

Install

tar -xzf thrity-network0.03.tar.gz
cd thrity-network
bash setup.sh
python3 browser/30web.py

Changelog

v0.03 latest download
  • Anonymize (Tor) is now per-tab, not browser-wide
  • Every tab fully isolated - separate cookies/login state per tab, not just no shared history
  • .thrity sites can now link to each other with thrity://name links, not just be typed in the address bar
  • New "Directory" button browses every site a registry knows about
  • Host server no longer exposes raw folder listings when a page is missing
  • Registry server now rate-limits registrations to slow down name-squatting scripts
v0.02 download
  • Tabs, no saved history — every tab is wiped on close
  • Optional HTTPS for .thrity sites (self-signed, auto-trusted)
  • "Anonymize" toggle routes traffic through Tor
  • Hardened WebKit settings (mic/camera blocked, no DNS prefetch, no local storage)
  • Fixed widespread TLS handshake errors on clearnet sites
v0.01 download
  • Address bar now shows the .thrity name instead of the real IP:port
v0.00 — first release download
  • 30web browser (WebKit2GTK), .thrity resolver, host server, registry server
